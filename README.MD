## Applied Blockchain eslint configuration

Usage:

Installation:

```bash
  npm install --save-dev @appliedblockchain/eslint-config
```

In your project's package.json

```json
  "eslintConfig": {
    "extends": "@appliedblockchain"
  }
```

## Linting

This project self-lints:

```bash
  npm i
  npm run lint
```
